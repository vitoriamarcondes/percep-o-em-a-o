## Upload no Wix

1. Subir todos os arquivos em `wix-studio/assets`.
2. Manter os nomes dos arquivos para facilitar a conexão com o CSV.
3. Em imagens de cards, usar `crop`/`cover`.
4. Evitar filtros escuros permanentes nas fotos. O site atual usa as imagens como sinal visual principal, não como textura genérica.
